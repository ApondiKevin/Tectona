GeoJSON

GeoJSON is a file format for representing map data. It is usually represented using the JSON(Javascript Object Notation) format. In GeoJSON, a vector feature and its attributes are represented as a JavaScript object, allowing for easy parsing of the geometry and fields.

GeoJSON is less bulky than XML-based structures such as KML; however, GeoJSON does not always contain styling information like KML does. You must define the styling on the client, which means writing JavaScript code.
